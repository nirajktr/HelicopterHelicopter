PROJECT TITLE:
ESCAPE THE DIDDY PARTY!

PURPOSE OF PROJECT:
Player controls Theodore, who is currently trying to escape Diddy's wrath. The game includes various features to enhance gameplay.

VERSION or DATE:
Version 1.0 - Novemeber 2024

HOW TO START THIS PROJECT:
1. Compile the Java files.
2. Run the main class to start the game.

AUTHORS:
Niraj Khatri

USER INSTRUCTIONS:
- Use the mouse to control the Theodore's movement.
- Press the mouse button to make Theodore go up.
- Release the mouse button to make Theodore go down.

FEATURES:
1. **Distance-Based Events**:
   - When the distance exceeds 100, a special sound (`special.wav`) is played.
   - The background color changes to red when the distance exceeds 100.

2. **Speed Adjustment**:
   - The speed of Theodore increases when certain distance thresholds are reached.

3. **Sound Effects**:
   - Sound effects are played for various events, such as starting the game and reaching distance milestones.

4. **Visual Changes**:
   - The background can change based on special game events.

Enjoy the game!
